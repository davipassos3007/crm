## 🚀 **Como Usar**

### **1. Instalação**
```bash
# Clone o repositório
git clone https://github.com/seu-usuario/crm-imobiliario

# Instale as dependências
cd crm-imobiliario
npm install

# Inicie o servidor
npm run dev
```

### **2. Primeiro Acesso**
1. Acesse http://localhost:5173
2. Use uma das contas demo ou crie sua conta
3. Explore o dashboard interativo
4. Teste o pipeline Kanban arrastando leads
5. Configure suas primeiras automações

### **3. Configuração Avançada**
```javascript
// Personalize cores no tailwind.config.js
colors: {
  'crm-primary': '#2563eb',
  'crm-secondary': '#1e40af',
  'crm-accent': '#3b82f6',
}

// Configure APIs no .env
VITE_WHATSAPP_API_KEY=sua_chave
VITE_SMS_API_KEY=sua_chave
VITE_EMAIL_API_KEY=sua_chave
```
